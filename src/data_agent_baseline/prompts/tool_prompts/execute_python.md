Run isolated Python over /context data and /scratch outputs; call set_answer to submit the final table. Direct source-preserving projection may call set_answer(columns, rows). Transformed answers should call set_answer(columns, rows, audit={...}) with source_paths and operations; output_row_count and output_hash are stamped automatically from the submitted table. If a prior tool saved an answer_candidate, the script has answer_candidate and read_answer_candidate() available; use that extracted table instead of re-parsing the original source. For PDF sources, use read_pdf_text('/context/file.pdf') or read_context_text('/context/file.pdf'); text-mode open('/context/file.pdf', 'r') also returns extracted text, while binary mode keeps the original bytes.
